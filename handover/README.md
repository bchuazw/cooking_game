# Hawker Mama Handover

This build is a clean Three.js voxel rebuild of the cooking game.

## Current Playable Build

- Hainanese Chicken Rice is the only dish currently exposed by `src/dishes/index.ts`.
- The dish uses a station workflow rather than separate dish-select mini-games: gather rice, cook rice, prep chicken, poach, pound chili sauce, plate, and serve.
- Rating and culture-card flow remain part of the end state.
- Old Laksa and Prata references are future-content drafts or historical notes, not currently playable routes.
