# Hawker Mama Handover

This build is a clean Three.js voxel rebuild of the cooking game.

## Current Playable Build

- Hainanese Chicken Rice
- Katong Laksa
- Roti Prata

Each dish has three mobile-first mini-games and a rating screen with a short cuisine-learning note.
